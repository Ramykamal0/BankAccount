# Bank Account System

A simple Bank Account Management System built with C# using Object-Oriented Programming (OOP) concepts.

## Features

* Create a bank account
* Deposit money
* Withdraw money
* Check account balance
* Display account information
* Demonstrates OOP principles

## Technologies Used

* C#
* .NET
* Object-Oriented Programming (OOP)

## OOP Concepts Applied

* Classes and Objects
* Encapsulation
* Inheritance (if implemented)
* Polymorphism (if implemented)
* Constructors
